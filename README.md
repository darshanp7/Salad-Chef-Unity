Player Controls:
-----------------------------------
<h4>Player 1:</h4> 

Movement Keys -> WASD 

Up: W, Down: S, Left: A, Right: D

Pick up item/Throw Item Key: E

Chop Vegetable Key: Q

<h4>Player 2:</h4>

Movement Keys -> Arrow Buttons

Up: Up, Down: Down, Left: Left, Right: Right

Pick up item/Throw Item Key: K

Chop Vegetable Key: L
