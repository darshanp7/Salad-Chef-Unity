Player Controls:
-----------------------------------
Player 1: 

Movement Keys -> WASD -> Up: W, Down: S, Left: A, Right: D
Pick up item/Throw Item Key: E
Chop Vegetable Key: Q

Player 2:
Movement Keys -> Arrow Buttons -> Up: Up, Down: Down, Left: Left, Right: Right
Pick up item/Throw Item Key: E
Chop Vegetable Key: Q
